심화 문제 — 코드 추적 & 개념 응용

14 / 15
맞힌 문제 0 / 0
HARD 01 — 출력값 직접 입력
다음 코드를 실행하면 어떤 값이 출력될까? 정확하게 입력하세요.
result = [] for i in range(1, 6): if i % 2 != 0: result.append(i * i) print(result)
홀수(i%2 != 0)만 골라서 i*i(제곱)을 리스트에 추가합니다. 1~5 중 홀수는 1, 3, 5.
홀수: 1→1², 3→3², 5→5² = 1, 9, 25 → 출력: [1, 9, 25]
HARD 02 — 예외처리 코드 추적
다음 코드의 출력값은?
data = ["10", "abc", "5"] result = [] for item in data: try: result.append(int(item) * 2) except ValueError: result.append(-1) print(result)
"10"→int 성공→20, "abc"→ValueError→-1, "5"→int 성공→10 → [20, -1, 10]
HARD 03 — 출력값 직접 입력
다음 코드를 실행하면 어떤 값이 출력될까?
d = {"a": 1, "b": 2} d["c"] = 3 d["a"] = 10 print(len(d), d["a"])
딕셔너리에 키가 이미 있으면 값이 덮어씌워집니다. 키가 새로 추가되면 항목 수가 늘어납니다.
시작: {a:1, b:2} → "c" 추가: {a:1, b:2, c:3} → "a" 덮어쓰기: {a:10, b:2, c:3}
len(d) = 3, d["a"] = 10
HARD 04 — 변수 범위 (Scope)
다음 코드를 실행하면 어떤 결과가 나타날까?
x = 5 def func(): x = x + 1 return x print(func())
함수 안에서 x = x + 1처럼 대입이 있으면 파이썬은 x를 지역변수로 봅니다.
하지만 오른쪽 x + 1을 평가할 때 지역변수 x가 아직 없어서 UnboundLocalError 발생.
해결: global x를 함수 맨 위에 선언하거나, 매개변수로 넘기기.
